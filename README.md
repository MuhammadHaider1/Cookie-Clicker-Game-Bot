# 🍪 Cookie Clicker Bot with Selenium (Python)

This project is a simple **automation bot** built with **Python + Selenium WebDriver** that plays the popular **Cookie Clicker** game automatically.

The bot opens the Cookie Clicker website, selects the **English language**, continuously clicks the big cookie to generate cookies, and every **5 seconds** it checks the store to buy the **most expensive available upgrade** it can afford.  
The script runs for **5 minutes**, then prints the final result.

---

## 🚀 Features

- Opens the Cookie Clicker game automatically
- Selects **English** language on startup
- Continuously clicks the **big cookie**
- Checks the store every **5 seconds**
- Buys the **best affordable product automatically**
- Runs for **5 minutes**
- Displays the **final cookie count** at the end

---

## 🛠️ Technologies Used

- **Python 3**
- **Selenium WebDriver**
- **Google Chrome**
- **ChromeDriver** (managed automatically if configured)

---

## 📂 Project Idea

This project demonstrates how to use **browser automation** with Selenium for:

- Web interaction
- Clicking buttons automatically
- Finding elements by **ID** and **CSS Selectors**
- Handling exceptions like `NoSuchElementException`
- Using timers and loops for automation logic

It’s a great beginner-friendly project for learning **Selenium automation** in a fun way.
